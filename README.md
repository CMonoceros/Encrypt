# Encrypt
本软件采用Android+Tomcat+MySQL体系。
前端采用Android系统，api最低要求为19即4.4，编译版本为23即6.0。为保证运行正常，建议使用6.0以上系统运行。
后端采用Tomcat服务器，测试版本为8.0，8.5，9.0。测试系统为CentOS6.5，CentOS7.0。
服务器采用MySQL5.7，测试系统为CentOS6.5，CentOS7.0。
